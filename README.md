# 📊 Retail-Sales-SQL-Pandas-Analysis

A Data Analysis portfolio project demonstrating end-to-end data cleaning and exploratory data analysis (EDA) using both **SQL** and **Python (Pandas)**.

## 🚀 Project Overview
This project simulates a real-world retail business case where raw data is messy and inconsistent. The same cleaning and analysis tasks are performed using two industry-standard tools for comparison.

### 📁 Repository Structure
-   **`data/`**: Contains raw, messy CSV datasets (`customers_raw`, `products_raw`, `orders_raw`).
-   **`SQL_Cleaning_Analysis.sql`**: Comprehensive SQL script covering data cleaning (CTEs, Row Numbers) and business questions (Joins, Aggregations).
-   **`Pandas_Cleaning_Analysis.py`**: Python script using Pandas for automated cleaning, analysis, and visualization.
-   **`Pandas_Cleaning_Results.png`**: Visual dashboard showcasing the insights.

## 🛠️ Key Skills Demonstrated
-   **SQL**: Advanced joins, CTEs, window functions, handling NULLs/Duplicates.
-   **Python**: Pandas (merging, cleaning, date formatting), Seaborn (visualization), NumPy.
-   **Data Engineering**: ETL processes and data standardization.
-   **Business Analysis**: Revenue growth tracking, customer LTV (Lifetime Value), and category performance analysis.

---
## 🏃 How to Run
1.  Ensure you have Python installed (`pip install pandas seaborn matplotlib`).
2.  Run the Python script to see the analysis:
    ```bash
    python Pandas_Cleaning_Analysis.py
    ```
3.  Open the SQL file in your preferred database client (SQLite compatible) to explore the query logic.
